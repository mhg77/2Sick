# 2Sick — украшения и аксессуары

Мобильное приложение iOS магазина украшений **2Sick**, написанное на SwiftUI.

## Возможности

- **Каталог** — просмотр коллекций и категорий украшений (серьги, колье, чокеры, кольца, браслеты, анклеты, сумки)
- **Детали товара** — фото, описание, выбор размера и количества
- **Избранное** — сохранение понравившихся изделий
- **Корзина** — управление заказом, подсчёт суммы
- **Кастомный заказ** — форма для создания украшения по индивидуальным параметрам

## Стек

| Технология | Версия |
|---|---|
| Swift | 6.0+ |
| SwiftUI | iOS 18+ |
| Архитектура | Observable (SwiftUI native state) |

## Структура проекта

```
2Sick/
├── Models.swift           — модели данных (Product, Cart, Order)
├── ShopStore.swift        — хранилище состояния приложения (@Observable)
├── Theme.swift            — цвета и стили бренда
├── CatalogView.swift      — экран каталога с фильтрами
├── ProductDetailView.swift — детальная карточка товара
├── FavoritesView.swift    — избранные товары
├── CartView.swift         — корзина и оформление заказа
└── CustomOrderView.swift  — форма кастомного заказа
```

## Запуск

1. Открыть `2Sick.xcodeproj` в Xcode 16+
2. Выбрать симулятор или устройство с iOS 18+
3. Запустить (⌘R)
